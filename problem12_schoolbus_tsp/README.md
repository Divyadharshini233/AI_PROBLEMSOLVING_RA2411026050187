# Problem 12: School Bus Route Optimization

## 📖 Description

Find an efficient route for a school bus to pick up students and return to school.

## 🧠 Algorithm

Nearest Neighbor Heuristic

## ⚙️ Input

* Stops
* Distance Matrix

## 📤 Output

* Route
* Total distance

## ▶️ Run

```bash
streamlit run app.py
```

## 📌 Note

Fast but may not always give optimal solution.
