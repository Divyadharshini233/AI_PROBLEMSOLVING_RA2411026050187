# Problem 7: Tourist Travel Planner (TSP)

## 📖 Description

Find the shortest route that visits all cities exactly once and returns to the start.

## 🧠 Algorithm

Brute Force (Checks all permutations)

## ⚙️ Input

* Cities
* Distance Matrix

## 📤 Output

* Optimal route
* Minimum distance

## ▶️ Run

```bash
streamlit run app.py
```

## 📌 Note

Guarantees best solution but slow for large inputs.
